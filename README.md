# Cipher Visualizer & Cracker

A playful, mobile-friendly web app to explore classic ciphers directly in the browser.  
No installation, no sign-up – everything runs locally.

## Features

- Caesar Cipher with automatic cracker
- Vigenère, Playfair, Polybius Square
- Atbash, ROT13, Affine, Rail Fence, Baconian, A1Z26
- Custom substitution with randomiser
- Dark/light theme toggle
- Clear explanations for every cipher

## Tech

Built with plain HTML, CSS, and JavaScript – no frameworks, no build step.
